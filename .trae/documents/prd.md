
## 1. Product Overview
PsyAssess 是一个 AI 优先的学校心理健康管理平台，旨在帮助心理老师高效管理学生心理健康测评、预警和干预工作。平台通过 AI 技术自动化量表生成、报告分析和异常检测，提高工作效率，及时发现学生心理风险。

## 2. Core Features

### 2.1 User Roles
| 角色 | 注册方法 | 核心权限 |
|------|---------|---------|
| 心理老师 | 系统账号 | 量表管理、测评任务、报告分析、预警处理 |
| 班主任 | 系统账号 | 查看班级学生状态、接收预警 |
| 学生 | 系统账号 | 在线测评、查看个人报告 |
| 管理员 | 系统账号 | 用户管理、系统配置、数据统计 |

### 2.2 Feature Module
1. **AI 量表生成页面**：输入描述自动生成专业心理测评量表
2. **量表管理页面**：量表的 CRUD、发布和禁用
3. **施测任务页面**：创建和管理测评任务，查看答题进度
4. **学生测评页面**：学生在线完成量表测评
5. **AI 报告生成页面**：自动生成个性化分析报告
6. **预警中心页面**：查看和处理心理风险预警
7. **学生档案页面**：查看和管理学生心理健康档案

### 2.3 Page Details
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| AI 量表生成页面 | 量表描述输入 | 用户输入量表类型、目标人群、评估维度等需求 |
| AI 量表生成页面 | AI 生成预览 | 展示 AI 生成的量表结构、题目、选项 |

## 3. Core Process
主要用户流程：心理老师创建 AI 量表 → 发布测评任务 → 学生在线答题 → AI 自动分析和预警 → 查看报告并处理预警。

```mermaid
flowchart TD
    A[心理老师] --&gt; B[AI生成量表]
    B --&gt; C[确认发布量表]
    C --&gt; D[创建施测任务]
    D --&gt; E[学生在线答题]
    E --&gt; F[AI自动评分和分析]
    F --&gt; G[生成报告]
    F --&gt; H[异常检测和预警]
    G --&gt; I[心理老师查看报告]
    H --&gt; J[心理老师处理预警]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：深蓝色（专业感）+ 柔和的青绿色（心理关怀感）
- 按钮风格：圆角、渐变填充、悬停阴影效果
- 字体：Inter（标题）+ Noto Sans SC（正文）
- 布局风格：卡片式布局，左侧导航栏，右侧内容区
- 图标风格：简洁的线性图标，来自 lucide-react/vue 库

### 4.2 Page Design Overview
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| AI 量表生成页面 | 描述输入区域 | 大文本框，提示文字，示例按钮 |
| AI 量表生成页面 | 生成预览区域 | 卡片式展示，可展开/收起维度和题目 |
| AI 量表生成页面 | 操作按钮 | 主要按钮（确认发布），次要按钮（重新生成） |
| 量表管理页面 | 量表列表 | 表格展示，状态标签，操作列 |
| 量表管理页面 | 搜索筛选 | 搜索框，分类筛选，状态筛选 |
| 施测任务页面 | 任务列表 | 进度条显示，状态标签，创建任务按钮 |
| 学生测评页面 | 答题界面 | 居中布局，题目卡片，进度指示 |
| AI 报告生成页面 | 报告预览 | 分段展示，图表可视化，下载按钮 |
| 预警中心页面 | 预警列表 | 严重级别颜色区分，时间排序，处理状态 |

### 4.3 Responsiveness
桌面优先设计，同时适配平板和移动设备，触摸优化，确保所有功能在不同设备上可用。

### 4.4 Performance Optimization
- 列表分页加载
- 图片懒加载
- 动画适度，避免过度渲染
